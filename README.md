## üöÄ Overview
**TKAWEN Titan ID** is a cutting-edge, decentralized digital identity and wallet connector for WordPress. Designed for EdTech platforms, B2B enterprises, and modern SaaS ecosystems, it transforms any standard WordPress site into a secure digital identity provider.

By leveraging advanced Google OAuth integration, dynamic QR code generation, and secure wallet architecture, Titan ID allows students and professionals to carry their verified identities directly on their mobile devices.

## ‚ú® Core Features
* **Sovereign Digital Wallet:** Every user is assigned a unique cryptographic Digital Identifier (DID).
* **Cinematic Mobile Experience:** A floating, responsive digital ID card optimized for mobile screens.
* **Frictionless Authentication:** Seamless Google Login (GSI) integration engineered to bypass aggressive server caching and strict security firewalls.
* **Integrated QR Scanner:** Built-in HTML5 QR scanner for instant, on-the-spot identity verification.
* **Academic Transcripts:** One-click printable academic records featuring digital signature hashing for maximum reliability.

## üõ†Ô∏è Installation
1. Download the latest release from the repository.
